# 宿舍管理系统 java1019

## 项目概述

宿舍管理系统是一款致力于提高宿舍管理效率与安全性的软件。该系统基于前后端分离的架构模式，后端采用Java语言结合SpringBoot框架，使用Mybatis作为持久层框架；前端则使用Vue.js构建响应式用户界面。数据存储方面，系统使用Mysql数据库进行结构化数据存储，同时使用Redis作为缓存数据库。

## 技术栈

- 后端：Java、SpringBoot、Mybatis
- 前端：Vue.js
- 数据库：Mysql、Redis

## 功能模块

### 系统管理

- 用户管理：对系统用户进行增删改查操作。
- 角色管理：分配与管理系统角色，并进行权限管理。
- 菜单管理：配置系统菜单结构，实现界面功能的个性化展示。
- 日志管理：记录系统操作日志，便于追踪与审计。

### 宿舍管理

- 宿舍楼管理：对宿舍楼的基本信息进行管理。
- 班级管理：维护班级与宿舍的分配关系。
- 宿舍管理：具体到宿舍单元的管理与维护。
- 学生管理：学生信息的管理，包括入住与迁出操作。

### 记录管理

- 维修记录：记录宿舍设施的维修情况。
- 晚归记录：记录学生的晚归情况，确保宿舍安全。
- 请假记录：管理学生的请假信息。

## 系统角色

系统中预设了管理员、宿舍管理员、教师和学生等角色，每个角色具有不同的权限与功能。

## 运行环境

- 后端：JDK 1.8及以上版本，运行在支持SpringBoot的服务器上。
- 前端：现代浏览器，支持HTML5和CSS3。
- 数据库：Mysql 5.7及以上版本，Redis服务。

## 部署步骤

1. 准备后端运行环境，包括Java虚拟机及相关的依赖库。
2. 部署Mysql和Redis数据库，并导入初始数据。
3. 部署前端Vue项目。
4. 配置服务器，启动后端服务。
5. 访问前端项目地址，进行系统登录。

## 目录结构

```
项目根目录/
│

├── frontend/               前端代码目录
└── database/               数据库文件目录
    ├── mysql/              Mysql数据库脚本
    └── redis/              Redis相关配置及数据
```
